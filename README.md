## 📖 О проекте

**NoteVision** — это программа для автоматического распознавания нот из изображений. Проект использует технологии компьютерного зрения для анализа нотных партитур и преобразования их в текстовый формат.

Просто загрузите изображение с нотами, и программа определит все ноты, их высоту и длительность, а затем выведет результат в буквенной или русской нотации.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🖼️ **Поддержка форматов** | JPG, PNG, BMP, TIFF |
| 🎵 **Распознавание нот** | Определение высоты и длительности |
| 📝 **Буквенная нотация** | C, D, E, F, G, A, B |
| 🇷🇺 **Русская нотация** | До, Ре, Ми, Фа, Соль, Ля, Си |
| 📄 **Экспорт** | TXT, JSON |
| 🎨 **Визуализация** | Выделение нот на изображении |
| 🚀 **Простой запуск** | Интерактивный выбор изображений |

## 🛠️ Технологии

- **Python 3.8+** — основной язык программирования
- **OpenCV 4.8.1** — компьютерное зрение и обработка изображений
- **NumPy 1.24.3** — работа с массивами и матрицами
- **Matplotlib 3.7.2** — визуализация результатов
- **Pillow 10.0.0** — работа с изображениями
